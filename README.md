This application needs a basic email/password login.

After login show a 2nd screen.
The screen will show the users current location on google maps.

The current users details will be added to the database
A 12 digit share code will created for each user.
This share code can be entered by an another user.
Application will then show the 2 users location on the google based map.
Diff coloured icons for each.

User should be able to enter and validate another users share code in a grey input field.


User should be able to select the icon of other users and stop tracking them.
